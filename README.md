# DRIL — VESTUÁRIO E REPETIÇÃO URBANA

> "Isso poderia existir colado num poste há 10 anos sem assinatura?"

Este repositório contém a estrutura digital da **DRIL**. Não é apenas um e-commerce; é um suporte para a ocupação do espaço simbólico através da streetart.

## 1. A MARCA (RESUMO DO MANUAL)
A DRIL não vende camisetas. Ela repete símbolos.
- **Território:** Streetart (Tag, Sticker, Lambe).
- **Estética:** Brutalista, cinza urbano, sem poesia.
- **Regra de Ouro:** Se precisar de texto para funcionar, falhou.

## 2. O PROJETO DIGITAL
O site foi construído para refletir a crueza da marca. 
- **Interface:** Minimalista, focada em tipografia sistemática.
- **Filtros:** Uso de `grayscale` e `contrast` via CSS para manter a unidade visual, independente da imagem.
- **Navegação:** Direta, sem distrações "instagramáveis" ou banners promocionais.

## 3. TECNOLOGIAS (LABORATÓRIO)
Este projeto é o ponto de partida de um plano de 5 anos, focado em:
- HTML5 Semântico.
- CSS3 (Variáveis, Grid, Sticky Header).
- Estética Brutalista / Anti-Design.

## 4. ESTRUTURA
- `index.html`: O esqueleto técnico.
- `style.css`: A camada de invisibilidade e peso gráfico.
- `/assets`: Repositório de símbolos (Pomba e variações).
